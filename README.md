# zhihu
一个仿知乎的小demo,前端：thymeleaf模板，后端：springboot+mybatis+redis+elasticsearch



###  功能

1. 登录/注册
2. 发表/评论问题
3. 用户相互关注
4. 关注问题
5. 问题或评论进行点赞点踩
6. 用户站内私信和通知
7. 全文检索
8. 问题敏感词过滤
